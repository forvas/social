Mail BCC

This module was written to add the field Blind Carbon Copy (BCC) to emails and email templates.

Installation

To install this module, you need to:

  • Check that you have installed the module email_template.

Usage

To use this module, you need to:

  • Go to Sales > Sales > Sales Orders, and create a new order.
  • Save the order and check that a notification email has been sent to all users who belong to Sales Manager group.
